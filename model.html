Introduction

Web development is a long term that is used for making the websites and web applications. The web development process include web-design and web content development. Mainly there are two types of web development: One is the front end which deals with all the design material using HTML,CSS, Javascript and it also deals with the UI. And the second one is the back end which deals with all the web system works.
A simple webpage can be made by using HTML5,CSS3 and normal Java script codes. We all were given a project to make a portfolio website and should only use HTML5,CSS3. The project contains 6 pages which are Home page,About me,CV,Social Media,Contact and Model_gp page.
To complete the project many website and designs were taken as a reference.
